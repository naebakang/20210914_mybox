2. 함수 이름의 첫 단어는 check, get, print, create, show, run 과 같은 6가지만 사용한다.
'함수는 1가지 기능을 해야한다.'는 원칙을 지키는 데 도움이 되며, 어떤 기능을 하는지
명확하게 알 수 있다.
2.1. check, True, False 와 같은 2 가지만 반환한다.
2.2. get, True, False 이 외의 값을 반환한다.
2.3. print, 특정 메세지를 출력한다.
2.4. create, 특정 확장자를 갖는 파일을 생성한다.
2.5. show, 특정 창을 출력한다.
2.6. run, 특정 프로그램을 실행시킨다.

3. 1개 class 당 1가지 책임(데이터 생성)을 맡게 한다. 추상적인 과정이며, 내가 이해하기 편하게 class 를 구성한다.

4. 값을 .xlsx, .pickle 등으로 저장하는 경우는 5가지이다.
4.1. 특정 주기 동안 초기화 되면 안되는 경우
4.2. 값을 얻기 까지 걸리는 시간이 600초(10분) 이상 일 경우
4.3. 오류를 찾는 데 결정적일 경우
4.4. 프로그램의 성능향상에 도움을 주는 경우
4.5. 외부에서 받아오는 데이터 중 비휘발성으로만 이용해야 하는 경우
